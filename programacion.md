## 15.¿Cuáles son los tipos de datos primitivos en Java? 
En Java existen ocho tipos de datos primitivos:

- byte: números enteros pequeños.
- short: números enteros cortos.
- int: números enteros.
- long: números enteros largos.
- float: números decimales de precisión simple.
- double: números decimales de doble precisión.
- char: un solo carácter.
- boolean: valores verdadero o falso (true / false).

## 16.¿Cómo funcionan las estructuras de control de flujo como if, else, switch y bucles en Java?
- if y else: ejecutan código según una condición verdadera o falsa.
- switch: evalúa una variable y ejecuta un bloque según el caso que coincida.
- Bucles (for, while, do-while): repiten un bloque de código mientras se cumpla una condición.  

Permiten controlar el flujo del programa según decisiones o repeticiones.

## 17.¿Por qué es importante usar nombres significativos para variables y métodos?
Porque mejora la legibilidad y comprensión del código.  
Nombres claros permiten entender qué hace una variable o método sin necesidad de muchos comentarios.  
Esto facilita el mantenimiento, el trabajo en equipo y evita errores.

## 18.¿Qué es la Programación Orientada a Objetos (POO)?
La Programación Orientada a Objetos es un paradigma de programación basado en la creación de objetos que representan entidades del mundo real.  
Estos objetos combinan datos (atributos) y comportamientos (métodos).

---
## 19.¿Cuáles son los cuatro pilares de la Programación Orientada a Objetos? 
- Encapsulamiento  
- Abstracción  
- Herencia  
- Polimorfismo  

Estos principios permiten organizar mejor el código y hacerlo más reutilizable y mantenible.

---
## 20.¿Qué es la herencia en POO y cómo se utiliza en Java? 
La herencia permite que una clase hija herede atributos y métodos de una clase padre.  
En Java se utiliza con la palabra clave extends.

Ejemplo:
class Hija extends Padre { }

Esto permite reutilizar código y crear relaciones entre clases.

## 21.¿Qué son los modificadores de acceso y cuáles son los más comunes en Java?
Los modificadores de acceso controlan la visibilidad de clases, atributos y métodos.

Los más comunes son:

- public: accesible desde cualquier parte.
- private: solo accesible dentro de la misma clase.
- protected: accesible dentro del mismo paquete o por herencia.
- (default): accesible solo dentro del mismo paquete.

## 22.¿Qué es una variable de entorno y por qué son importantes para Java o la programación en general?
Una variable de entorno es un valor configurado en el sistema operativo que puede ser usado por programas.

En Java son importantes para:
- Configurar el JAVA_HOME.
- Definir rutas del sistema (PATH).
- Manejar configuraciones sin modificar el código.

Permiten que los programas funcionen correctamente en diferentes entornos.
